**2. Figury przestrzenne**
==========================

Figury przestrzenne (bryły) mają trzy wymiary: długość, szerokość i
wysokość. Ich objętość i pole powierzchni są kluczowe w wielu
zastosowaniach. Do najczęściej spotykanych należą:

+-------+-------+---------------------------+------------------------+
|       | *     | **Sześcian**              | wszystkie ściany są    |
|       | *1.** |                           | kwadratami, a          |
|       |       |                           | krawędzie mają równą   |
|       |       |                           | długość.               |
+=======+=======+===========================+========================+
|       | *     | **Prostopadłościan**      | ma sześć ścian         |
|       | *2.** |                           | prostokątnych, często  |
|       |       |                           | używany do opisu       |
|       |       |                           | pudełek lub budynków.  |
+-------+-------+---------------------------+------------------------+
|       | *     | **Kula**                  | zbiór wszystkich       |
|       | *3.** |                           | punktów w przestrzeni  |
|       |       |                           | równo oddalonych od    |
|       |       |                           | środka.                |
+-------+-------+---------------------------+------------------------+
|       | *     | **Stożek i walec**        | mają podstawy w        |
|       | *4.** |                           | kształcie koła, różnią |
|       |       |                           | się jednak sposobem    |
|       |       |                           | łączenia z             |
|       |       |                           | wierzchołkiem lub      |
|       |       |                           | drugą podstawą.        |
+-------+-------+---------------------------+------------------------+
